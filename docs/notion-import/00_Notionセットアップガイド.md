# ぺいほーむ Notionワークスペース セットアップガイド

## インポート手順

### 方法1: コピペ（推奨・最速）

1. Notionで新規ページを作成
2. 各mdファイルの内容をコピー
3. Notionページに貼り付け → 自動的にNotion形式に変換される
4. データベース部分は「/database」コマンドで作成し、CSVインポート

### 方法2: ファイルインポート

1. Notion左サイドバー「Import」をクリック
2. 「Markdown & CSV」を選択
3. `docs/notion-import/` フォルダ内のmdファイルを選択
4. インポート完了

## 推奨ワークスペース構成

```
ぺいほーむ (トップページ)
├── 📋 タスク管理DB (データベース)
├── 📊 ダッシュボード (ページ)
├── 📂 ドキュメント
│   ├── 01_エンジニア共有用
│   ├── 02_営業関係
│   ├── 03_経理関係
│   ├── 04_契約書
│   ├── 05_事業計画・戦略
│   ├── 06_業務運用ルール
│   └── 07_新規整備
├── 🔄 業務フロー (ページ)
├── 📅 リリース当日進行表 (ページ)
└── ⚙️ 運用ルール (ページ)
```

## インポート順序

1. **最初**: `01_ダッシュボード.md` → トップページに
2. **次**: `02_タスク管理DB.csv` → データベースとしてインポート
3. **その後**: 残りのmdファイルを順番にインポート
