## 3. ロールバック手順

### Vercel GUI 経由 (推奨)

1. Vercel ダッシュボード → Project → Deployments
2. 直近の正常デプロイ (`release/mvp-2026-05` 以前の main 最終コミット等) を選択
3. **"Promote to Production"** をクリック
4. 30 秒〜1 分で本番が切り戻る
5. Slack で全員に通知

### Git revert 経由

```bash
git checkout main
git revert <問題のあるコミットハッシュ>
git push origin main
# Vercel が自動デプロイ
```
