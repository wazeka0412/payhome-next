## 実装メモ (ぺいほーむ側の手順)

### 自動化のための設計

Phase 4 (2026/12〜) で SaaS プランの一部として自動配信を実装する予定:

1. Supabase の集計クエリで工務店ごとのデータを集計
2. Markdown テンプレートに値を埋め込み
3. PDF 生成 (md → pdf)
4. Resend 経由でメール配信 (送信先: 工務店担当メール)
5. ダッシュボード `/dashboard/builder/reports` で閲覧可能

### Phase 1〜3 の暫定対応

Phase 4 以前は、PM が手動で以下の手順で配信:

1. 月初 3 営業日以内に本テンプレートをコピー
2. Supabase で工務店別の集計クエリを実行 (`scripts/monthly-report-queries.sql`)
3. 各値を埋める
4. PDF 化 (Pandoc or Google Docs)
5. メール送信
