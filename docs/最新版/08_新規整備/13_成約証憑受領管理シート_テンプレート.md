## 1. このシートの目的

Phase 3 以降の成果報酬 3% 運用において、工務店からの成約報告と証憑受領を追跡し、請求漏れ・不正防止・入金管理を行うためのシート。

## 3. ステータス遷移

```
reported (成約報告受領)
   ↓ ぺいほーむ側で対象判定
verified (対象確認済み) or rejected (対象外)
   ↓ 対象なら月末集計
invoiced (請求書発行済み)
   ↓ 30日以内
paid (入金確認済み) or overdue (支払遅延)
   ↓
closed (完了)
```

**disputed**: 工務店側と認識相違がある場合

## 5. 運用ルール

### 5.1 入力タイミング

- **工務店が成約報告した当日**: `reported` ステータスで入力
- **対象判定後 48 時間以内**: `verified` または `rejected` に変更
- **月末集計時**: `invoiced` に変更
- **入金確認後**: `paid` に変更

### 5.2 対象判定ロジック

`is_target = TRUE` の条件:

1. `lead_source ∈ { consultation, diagnosis, member }`
2. `contract_signed_date - lead_received_date ≤ 6 months`
3. 重複成約がない (同じ user_id が他の工務店で成約していない)
4. 既存顧客フラグが FALSE

### 5.3 証憑保管

- `evidence_file` に記載されたファイル名は `docs/経理/証憑/YYYY-MM/[工務店名]/[user_name].pdf` に対応
- 原本は Supabase Storage にアップロード
- 7 年間保管 (法人税法準拠)

### 5.4 監査クエリ

月次で以下を実行し、不整合を検出:

```sql
-- 未報告の疑いがある成約 (工務店への送客から長期間経過)
SELECT lead_id, user_name, builder_name, lead_received_date,
       EXTRACT(day FROM now() - lead_received_date) AS days_since_received
FROM leads
WHERE status IN ('event_booked', 'event_attended')
  AND lead_source IN ('consultation', 'diagnosis', 'member')
  AND contract_signed_date IS NULL
  AND lead_received_date > now() - INTERVAL '6 months'
ORDER BY lead_received_date;

-- 対象判定漏れの疑い (ぺいほーむ側で見逃している可能性)
SELECT *
FROM success_fee_ledger
WHERE status = 'reported'
  AND report_date < now() - INTERVAL '7 days';
```

## 6. 月次集計の方法

### 手順

1. 前月の `status = 'verified'` を全件抽出
2. 工務店ごとに `success_fee` を合計
3. `請求・P&L管理.xlsx > ① 月次請求台帳` に記録
4. freee で請求書作成
5. 送付時に `status = 'invoiced'` に更新

### 請求書記載の元データ

```sql
SELECT
  builder_name,
  COUNT(*) AS success_count,
  SUM(contract_amount) AS total_contract,
  SUM(success_fee) AS total_success_fee
FROM success_fee_ledger
WHERE status = 'verified'
  AND invoice_month = '2026-09'
GROUP BY builder_name;
```

## 7. 入金管理の運用

### 入金確認

- 毎週月曜に経理がネットバンキングで確認
- 入金があれば該当行を `status = 'paid'` に更新
- 入金日・入金額を記録

### 支払遅延のエスカレーション

- 支払期限 +7 日 経過で督促メール
- +14 日 経過で PM に通知
- +30 日 経過で代表に通知 → 電話督促

- 工務店別の成約率
- 報告遅延の傾向
