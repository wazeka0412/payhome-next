## docx/xlsx への変換

これらの資料は markdown で作成されています。Word/Excel 形式で配布する場合:

```bash
# docx 生成
python3 scripts/convert-latest-md-to-docs.py
# (要: scripts/convert-v3-docs.py のような変換スクリプト)
```

あるいは Pandoc で:

```bash
for f in docs/最新版/08_新規整備/*.md; do
  pandoc "$f" -o "${f%.md}.docx"
done
```
