# お家づくりカルテ 要件定義・実装仕様書

最終更新: 2026-04-12
対象: PM / エンジニア / 営業 / CS

---

## 1. 目的

ヒアリングシートの情報を、会員登録後に段階的に蓄積する「お家づくりカルテ」として再設計し、以下を実現する。

- ユーザーが自分の希望を言語化・整理できる
- AI診断の推薦精度が向上する
- 工務店への商談前要約が強化される
- 成果報酬50万円の「伴走・教育・意向整理」の証拠が積み上がる
- 見学予約・無料相談の質が向上する

---

## 2. AI診断との違い

| | AI診断 | お家づくりカルテ |
|---|--------|---------------|
| **目的** | 方向性の整理（2分で完了） | 理想と条件の蓄積（少しずつ育つ） |
| **所要時間** | 2分・10問 | 5分〜（段階的に更新） |
| **結果** | おすすめ工務店3社 | 自分の条件整理＋商談前要約 |
| **更新** | 1回きり | 何度でも更新可能 |
| **位置づけ** | 核ループの入口 | 核ループを太くする補強機能 |

## 5. 商談前要約仕様

工務店に渡す1ページ要約。以下の構成：

```
━━━━━━━━━━━━━━━━━━━━━━━
商談前要約 — ○○様
━━━━━━━━━━━━━━━━━━━━━━━

【基本情報】
・家族構成: 本人(30) + 婚約者(31)
・検討開始: 2025年1月
・入居希望: できるだけ早く
・検討温度感: かなり具体的

【予算】
・総予算: 3,800万円（建物3,000万 + 土地800万）
・自己資金: 300万円 / 月返済希望: 7-9万円
・⚠️ 自動車ローン2.7万/月あり

【土地・エリア】
・計画地なし → 検討エリア: 国分上小川 / 国分広瀬 / 国分福島
・希望敷地: 70-80坪

【間取り・広さ】
・延床: 30-35坪 / 平屋 / 4LDK
・駐車: 3台分（所有2台）

【デザイン・性能】
・シンプルモダン（工業製品メイン、自然素材強すぎはNG）
・南面LDK横長配置希望
・重視: 耐震性 / 2×4工法

【比較状況】
・ニューイングホーム ★★★★☆ 最有力（OB紹介）
・NEO DESIGN HOME ★★★★☆ ぺいほーむ紹介
・アイ工務店 ★★★☆☆ 4/5顔合わせ
・セキスイハイム ★★★☆☆ 高すぎ保留
・ケイエイツー — お断り（営業対応問題）

【⚠️ 営業対応 重要】
✅ 求めること: 親身さ / レスポンス速さ / 一貫体制 / 視覚的提案
❌ NG: しつこい営業 / 突然の自宅訪問 / 過度な電話
📝 連絡は当日アポ確認程度でOK

【動画で気に入った点】
・国分松木の完成見学会が理想に近い

【見学で確認したいこと】
・職人は毎回同じ人か / 性能の他社比較の強み
```

## 6. 成果報酬50万円モデルへの接続

カルテの各STEPが成果報酬条件を補強：

| STEP | 成果報酬条件との対応 |
|------|---------------------|
| STEP 0 | 会員登録の証拠 |
| STEP 1 | AI診断完了の証拠 |
| STEP 2-3 | 教育導線通過（条件整理=意思決定支援） |
| STEP 4 | 動画コンテンツ活用の証拠 |
| STEP 5 | 比較整理・営業相性設計の証拠 |
| STEP 6 | 見学予約前の伴走支援の証拠 |

**カルテ入力率が高い = ぺいほーむが関与した証拠が厚い**

## 7. データ設計

テーブル: `home_building_profiles` (006_home_building_profiles.sql)

- user_id (UUID, UNIQUE) — 会員1人1レコード
- STEP 0〜6の全項目をカラムで保持
- comparing_companies は JSONB（柔軟な配列）
- completion_rate (INTEGER) — 入力率自動計算
- staff_memo (TEXT) — ぺいほーむ担当者メモ

API: `POST/GET /api/profile/home-building`
