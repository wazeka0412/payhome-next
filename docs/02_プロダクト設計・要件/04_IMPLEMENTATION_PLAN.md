## 🔥 v4.0 戦略方針（2026-04-02）

工務店ヒアリングにより「資料請求は質が低く成約に繋がらない」ことが判明したため、実装優先順位を **資料請求中心 → 会員登録 × AI診断 × 見学会予約** の高品質送客モデルへ転換する。

### ⛔ 廃止・縮小する機能
- **F-07：まとめて資料請求** → 廃止（実装しない）
- 資料請求フォーム → 存在は残すが、ナビ / CTAから除外
- 月刊 / ウェビナー / ニュース / コラム → MVPから除外

### 最優先（5月リリース前）
- **会員登録フロー**（メール / Google OAuth / マジックリンク）
- **AI家づくり診断**（10問 + 結果保存 + 工務店レコメンド）
- **動画詳細ページの見学会予約セクション**（モデルハウス時）
- **工務店個別ページのCTAを見学会予約に統一**
- **簡易企業管理画面MVP**（リード受信 / プロフィール / 見学会管理の3画面）
- **会員限定コンテンツのブラー化 + 登録CTA**
- **50社分の実データ投入**（全収益の前提）
- F-08：見学会残席表示（シンプル版）

### 中優先（Phase 1.5 = リリース後1〜2ヶ月）
- AI予約前ヒアリング整理（リード詳細にAI要約表示）
- AIリードスコアリング（静的 → 動的）
- 平屋事例ライブラリ MVP（会員限定）
- 家づくりコミュニティ MVP（会員限定）
- GA4カスタムイベント実装
- AI会話要約・意図抽出バッチ

### 後回し（Phase 2 = 3〜6ヶ月）
- 月刊ぺいほーむ / ウェビナー / ニュース / コラム復活
- 工務店ダッシュボードのフル機能
- AI営業支援機能
- LINE連携通知
